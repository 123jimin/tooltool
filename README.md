# tooltool

This library provides various small utilities for TypeScript projects.

Currently, tooltool is my personal utility library, and no backward compatibility is guaranteed.

## Features

- No dependencies!
- Supports both Node.js (22+) and modern web browsers.

## Usage

```bash
npm install @jiminp/tooltool
```

To see which utilities are available, [check the documentation](https://123jimin.github.io/tooltool).
